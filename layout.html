레이아웃 실습 2

포지셔닝

콘텐츠의 위치를 지정하는 방법으로 크게 5가지의 속성을 가질 수 있다.

  • position: static [일반적인 흐름에 따라 요소를 배치]
  • position: absolute [절대적인 위치를 지정]
  • position: relative [상대적인 위치를 지정]
  • position: fixed [고정적인 위치에 배치]
  • position: sticky(아직 표준x) [static+absolute]

플로팅(float)

콘텐츠의 위치를 지정하는 방법으로 좌,우로만 배치할 수 있음. 또한, 한번 float을 사용하면 언젠가 clear 해야 함.
